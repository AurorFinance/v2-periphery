# Local Development

The following assumes the use of `node@10.24.1`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

## Deployment

`cd deployment && npx hardhat run scripts/deployRouter.ts`
